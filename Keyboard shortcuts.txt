Blender Keyboard Shortcuts
General Navigation:
- Middle Mouse Button (MMB): Orbit the view.
- Shift + MMB: Pan the view.
- Scroll Wheel: Zoom in/out.
- Numpad 0: Camera view.
- Numpad 1: Front view.
- Numpad 3: Side view.
- Numpad 7: Top view.
- Numpad 5: Toggle perspective/orthographic view.
Object Mode:
- G: Grab/move.
- S: Scale.
- R: Rotate.
- X/Y/Z: Constrain movement/scale/rotate to axis.
- Shift + X/Y/Z: Constrain to two axes.
- Tab: Switch between Object and Edit mode.
- Ctrl + A: Apply transformations.
- Ctrl + J: Join selected objects.
- Ctrl + L: Link objects' data.
- Ctrl + D: Duplicate.
- Shift + D: Duplicate and move.
- Alt + D: Create linked duplicate.
Blender Keyboard Shortcuts
- M: Move to collection.
Edit Mode:
- Ctrl + R: Loop cut.
- E: Extrude.
- I: Inset face.
- Alt + E: Extrude along normals or individual faces.
- Ctrl + B: Bevel edges.
- K: Knife tool.
- Alt + M: Merge vertices.
- Shift + R: Repeat last action.
- F: Fill face/edge.
- Ctrl + E: Access edge menu.
Sculpt Mode:
- F: Adjust brush size.
- Shift + F: Adjust brush strength.
- Shift: Smooth while sculpting.
- Ctrl: Invert brush action.
- X/Y/Z: Symmetry along an axis.
Rendering and Camera:
- F12: Render image.
- Shift + F12: Render animation.
Blender Keyboard Shortcuts
- Ctrl + B: Set render border.
- Ctrl + Alt + Numpad 0: Align camera to view.
- Ctrl + Shift + B: Clear render border.
Timeline/Animation:
- Space: Play/Pause animation.
- Right Arrow: Next frame.
- Left Arrow: Previous frame.
- Shift + Left/Right Arrow: Jump to keyframes.
- Ctrl + Space: Maximize/Minimize current editor.
Shader Editor:
- Shift + A: Add node.
- Ctrl + Left Click + Drag: Connect nodes.
- Shift + D: Duplicate node.
- Alt + Right Click + Drag: Cut node connection.
- G: Move node.
- T: Show/Hide toolbar.
- N: Show/Hide properties panel.
File Management:
- Ctrl + S: Save.
- Ctrl + Shift + S: Save As.
- Ctrl + O: Open.
Blender Keyboard Shortcuts
- Ctrl + N: New file.
- Ctrl + Z: Undo.
- Ctrl + Shift + Z: Redo.
Additional Shortcuts:
- Shift + Space + [Tool Key]: Access specific tools like grab, rotate, scale, etc.
- Alt + H: Unhide all hidden objects.
- H: Hide selected object/element.
- Shift + H: Hide unselected objects/elements.
- Ctrl + Alt + Q: Quad view.
